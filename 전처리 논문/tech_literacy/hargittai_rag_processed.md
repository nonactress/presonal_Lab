- 도메인: 일반 인터넷 사용자 (웹 서비스 타겟 사용자 전반에 적용 가능)

---

# CHUNK 1 — 핵심 주장: "디지털 네이티브도 스킬이 다 다르다"
**검색 키워드 힌트:** 사용자 스킬 차이, 디지털 리터러시, 초보 사용자

인터넷을 오래 써온 젊은 사용자도 디지털 스킬이 균일하지 않다.
단순히 인터넷 접근성(access)이 있다고 해서 능숙하게 사용하는 것이 아니며,
같은 연령대·학력 집단 내에서도 스킬 수준은 크게 벌어진다.

→ **페르소나 적용:** 타겟 사용자가 "젊은 층" 또는 "인터넷 사용자"라는 이유만으로
능숙한 사용자로 가정하면 안 된다. 스킬 수준은 별도 변수로 고려해야 한다.

---

# CHUNK 2 — 사용자 스킬 수준을 결정하는 요인들
**검색 키워드 힌트:** 사용자 특성, 스킬 수준 예측, 페르소나 변수

다음 요인들이 사용자의 웹 사용 스킬 수준과 연관된다:

1. **사회경제적 배경 (SES):** 부모 학력이 높을수록 스킬 수준도 높은 경향
2. **성별:** 남성이 여성보다 평균적으로 높은 스킬 점수를 보고 (자기보고 기준)
3. **인터넷 자율성 (Autonomy of use):** 원하는 때/장소에서 자유롭게 쓸 수 있는 환경일수록 스킬 높음
4. **사용 경험 연수:** 오래 사용할수록 스킬 높음 (단, 일정 수준 이상에서는 수확 체감)
5. **주간 사용 시간:** 많이 쓸수록 스킬 높음

→ **페르소나 적용:** 타겟 사용자의 SES, 성별, 인터넷 환경, 사용 연차를 알면
스킬 수준을 간접 추정할 수 있다.

---

# CHUNK 3 — 스킬 수준이 실제 행동(사용 다양성)에 미치는 영향
**검색 키워드 힌트:** 스킬과 행동 패턴, 사용자 행동 예측, UX 막힘 지점

스킬이 높은 사용자는 인터넷을 더 다양한 목적으로 사용한다.
스킬이 낮은 사용자는 익숙한 기능만 반복 사용하고, 새로운 기능 탐색에 어려움을 겪는다.

스킬 변수는 성별·학력보다 사용 다양성을 더 잘 설명하는 매개 변수로 작동한다.
즉, 배경 특성이 행동 차이로 이어지는 경로의 핵심에 "스킬"이 있다.

→ **페르소나 적용:**
- 스킬 낮은 사용자 페르소나 → UI 탐색 어려움, 새 기능 미사용, 핵심 flow 이탈 가능성 높음
- 스킬 높은 사용자 페르소나 → 다양한 기능 시도, 빠른 탐색, 단축키/고급 기능 기대

---

# CHUNK 4 — 인터넷 자율성(Autonomy)과 사용 환경의 중요성
**검색 키워드 힌트:** 사용 환경, 접근 맥락, 모바일 사용자, 제약 상황

자율성(autonomy of use)은 원하는 시간과 장소에서 자유롭게 기술을 사용할 수 있는 정도를 의미한다.
자율성이 높을수록(다양한 접근 경로 보유, 본인 소유 기기 사용 등) 스킬과 사용 다양성 모두 높아진다.

→ **페르소나 적용:**
- 공용 PC 사용자, 모바일만 사용하는 사용자, 데이터 제한 환경의 사용자는
  자율성이 낮아 서비스 탐색 깊이가 낮을 수 있다.
- 온보딩 단계에서 이탈 가능성이 높은 집단으로 예측 가능.

---

# CHUNK 5 — "capital-enhancing" vs 오락적 사용 패턴
**검색 키워드 힌트:** 사용 목적, 사용자 동기, 정보 탐색형 vs 오락형

사용자의 온라인 활동은 두 유형으로 구분할 수 있다:
- **Capital-enhancing 활동:** 건강 정보 검색, 금융 거래, 뉴스, 취업 정보 등 — 실질적 이득과 연관
- **오락적 활동:** 게임, 음악 다운로드, 목적 없는 브라우징 등

학력·SES가 높은 사용자는 capital-enhancing 활동에 더 많이 참여하는 경향이 있다.

→ **페르소나 적용:**
- 서비스 목적이 정보 제공/생산성 도구라면 → 스킬·SES 높은 사용자가 더 적극 활용
- 엔터테인먼트/소셜 서비스라면 → SES와 관계없이 폭넓은 사용자 유입 가능
- 두 유형의 사용자가 "같은 기능에서 다른 이유로 막힐 수 있음"을 고려해야 함

---

# CHUNK 6 — 스킬 자기인식 vs 실제 스킬의 괴리 (특히 성별)
**검색 키워드 힌트:** 사용자 자신감, 자기효능감, 여성 사용자, 초보 사용자

여성 사용자는 실제 스킬이 통제되어도 스킬에 대한 자기평가를 낮게 하는 경향이 있다.
즉, 실제로는 할 수 있는데 "나는 잘 모른다"고 느끼는 경우가 더 많다.

이 자기인식의 차이는 실제 행동(사용 다양성, 콘텐츠 공유 등)에 영향을 미친다.

→ **페르소나 적용:**
- 여성 사용자 또는 자기효능감이 낮은 사용자 페르소나는 실제 기능보다
  "할 수 있을 것 같지 않아서" 시도조차 안 하는 패턴을 보일 수 있다.
- UX 관점에서 이 집단은 에러 메시지·복잡한 UI에서 조기 이탈할 가능성이 높다.
