# C1 — 데이터 소스 (MVP: 공공 데이터 & 논문 중심)

## MVP 원칙
**"2주 내 구축을 위해 크롤링 이슈가 있는 커뮤니티 데이터는 제외하고, 검증된 공공 데이터와 논문 RAG에 집중한다."**

---

## 데이터 소스 우선순위

### 1순위 — 공공 데이터 (직접 수치화 가능)
- **디지털정보격차 실태조사 (과기정통부):** 연령별 디지털 역량 지수 (`digital_literacy`)
- **한국미디어패널조사 (KISDI):** 매체 이용 행태 (`ui_pattern_familiarity`)

### 2순위 — UX 연구 논문 (심리적 근거)
- 노인/장애인 UX 가이드라인 논문
- 한국인 특유의 디지털 행동 패턴 연구 (빨리빨리, 외부 의존성 등)

---

## MVP에서 제외되는 것
- **커뮤니티 크롤링:** 에브리타임, 네이트판 등 (계정 차단 및 파싱 복잡성으로 인해 MVP 이후로 연기)
- **실제 유저 Think-Aloud 수집:** 데이터 확보 시간이 부족하므로 MVP 이후 "데이터 보정" 단계에서 수행

---

## 데이터 RAG 구축 방식
1. 공공 데이터 보고서(PDF) 및 통계표(Excel) 확보
2. 핵심 수치(예: 60대의 모바일 뱅킹 이용률)를 JSON 형태로 정형화
3. 논문 텍스트를 ChromaDB에 적재하여 PersonaParams 생성 시 근거로 활용
