# PersonaLab — 2-Week MVP (Code Analyzer Pivot)

## 🚀 MVP 핵심 목표
**"개발자가 작성 중인 프론트엔드 코드를 넣으면, 타겟 유저 입장에서 UX를 분석하고 즉시 고칠 수 있는 'Vibe Coding 프롬프트'를 제공한다."**

- **타겟:** Cursor, v0, Windsurf 등을 사용하는 AI 협업 개발자
- **기한:** 2주
- **핵심 가치:** 브라우저를 띄우기 전, 코드 단계에서 발견하는 'UX Linter'

---

## 2주 개발 일정 (MVP)

### Week 1: 코드 분석 파이프라인 구축
- **Day 1-2:** M1 코드 입력 시스템 및 LLM 'Mental Rendering' 프롬프트 설계
- **Day 3-4:** M2 공공 데이터 RAG 구축 (페르소나 파라미터화)
- **Day 5-7:** M1+M2+M3 통합 (코드 -> 페르소나 반응 추출)

### Week 2: 결과물 고도화 및 UI 완성
- **Day 8-9:** M4 **라인 번호 기반** Fix Prompt 생성 로직 최적화
- **Day 10-12:** M5 Gradio UI (Code Editor 스타일 입력 + 결과 카드)
- **Day 13-14:** 최종 테스트 및 데모 시나리오 (예: '노인용 약 알림 앱 컴포넌트') 완성

---

## MVP 기술 스택
- **AI:** GPT-4o API (코드 분석 및 시뮬레이션)
- **UI:** Gradio (코드 입력창 `gr.Code` 지원)
- **Data:** 공공데이터(디지털정보격차 실태조사) + UX 논문 PDF
- **No Playwright:** 브라우저 자동화 의존성 제거로 가볍고 빠른 실행
